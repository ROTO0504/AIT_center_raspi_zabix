# ライトの仕様

## GPIO 側の仕様

| GPIO | 配線色   | ライトの先 |
| ---- | -------- | ---------- |
| 6    | 緑       | LED 緑     |
| 13   | 黄       | LED 黄     |
| 19   | オレンジ | ブザー     |
| 26   | 赤       | LED 赤     |
| GND  | 茶       | 共通の GND |

## ライトの動作(zabbix のメールに基づく)

メール受信時は本文を判定し、以下のルールでライトとブザーを動作させる。

- `high` 以上を検出: **赤 LED 点灯 2 回鳴らす 0.2 秒空けて**

- 起動時は黄色 LED 点滅

## ボタンの仕様

一回押すと、ライトを消灯・ブザーを 0.5 秒一回鳴らす。
ステータスは戻さない。

## 起動方法（簡易）

以下の 1 コマンドで、メール受信監視・Web 反映・GPIO 制御・ボタン監視をまとめて起動できる。

```bash
python ver2/start_system.py
```

オプション:

- 依存インストールを省略: `python ver2/start_system.py --skip-install`
- ポート指定: `python ver2/start_system.py --port 8081`

### 常時確認の設定

ラズパイ常時稼働でメールを高頻度に反映する場合は、`.env` に以下を設定する。

- `MAIL_POLL_INTERVAL=5` （5 秒ごとにメール確認）

※ 旧設定 `POLL_INTERVAL` は後方互換として利用可能。
